API Documentation

1. Fetch DNS Records

POST /api/dns_lookup

{
  "website": "example.com"
}

Response:

{
  "A": ["192.0.2.1"],
  "MX": ["mail.example.com"],
  "NS": ["ns1.example.com"]
}

2. Reverse DNS Lookup

POST /reverse_dns

{
  "ip": "192.0.2.1"
}

Response:

{
  "domain": "example.com"
}

3. Domain Health Check

POST /domain_health

{
  "website": "example.com"
}

Response:

{
  "DNS Resolution": "Domain resolves successfully.",
  "Blacklist Status": "No issues found",
  "DNSSEC": "DNSSEC status not implemented yet."
}
